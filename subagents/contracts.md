## 1. EvidenceItem
```json
{
  "id": "S1",
  "title": "来源标题",
  "url": "https://...",
  "level": "一级|二级|三级",
  "publisher": "发布主体",
  "publish_date": "YYYY-MM-DD",
  "access_checked_at": "YYYY-MM-DD",
  "subject_verified": true
}
```

## 3. 宏观行业研判输出
```json
{
  "宏观清单": {
    "风险": ["FactRow x >=8"],
    "利好": ["FactRow x >=8"]
  },
  "行业深研": {
    "政策风向": {"结论": "...", "证据来源": ["S1", "S2"]},
    "行业景气度": {"结论": "...", "证据来源": ["S3"]},
    "未来增值空间": {"结论": "...", "证据来源": ["S4"]}
  },
  "基本面分析检查": {
    "收入结构": {"结论": "...", "证据来源": ["S5"], "日期": "YYYY-MM-DD", "统计口径": "TTM|单季|累计"},
    "利润质量": {"结论": "...", "证据来源": ["S6"], "日期": "YYYY-MM-DD", "统计口径": "TTM|单季|累计"},
    "现金流质量": {"结论": "...", "证据来源": ["S7"], "日期": "YYYY-MM-DD", "统计口径": "TTM|单季|累计"},
    "估值与预期": {"结论": "...", "证据来源": ["S8"], "日期": "YYYY-MM-DD", "统计口径": "TTM|单季|累计"},
    "催化财务映射": {"结论": "...", "证据来源": ["S9"], "日期": "YYYY-MM-DD", "统计口径": "TTM|单季|累计"}
  },
  "sources": {"S1": "EvidenceItem", "S2": "EvidenceItem"}
}
```

## 4. 基本面分析检查标准
- 固定维度必须完整：`收入结构`、`利润质量`、`现金流质量`、`估值与预期`、`催化财务映射`。
- 每个维度必须包含：`结论`、`证据来源`、`日期`、`统计口径`。
- 每个维度必须至少关联一个 `sources` 编号。
