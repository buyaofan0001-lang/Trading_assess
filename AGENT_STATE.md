The UI has four modules: today's permission gate, dynamically generated holding-vs-peer arena with intraday charts and embedded overseas peers, post-close sector/concept flow, and the Markdown journal editor. Start with `python3 dashboard/server.py`, then open `http://127.0.0.1:8765`.
