## 二、未来10个交易日的恢复期

### 第1—5个交易日

- 禁止给亏损仓加仓；长电科技不得继续补仓。

- 初始仓不超过账户权益的10%。

## 三、不可协商的交易规则

1. **生活费与交易本金隔离**：房租、应急金和未来至少12个月基本生活费不得放进交易账户；在非交易现金流不稳定前，不追加交易本金。
2. **禁止向下加仓**：任何新增仓位必须在原持仓已有至少5%收盘浮盈后发生，单次加仓不超过首仓的50%。
3. **买入时间闸门**：14:30以前不得主动买入。券商App或手机专注模式应在14:25前限制下单入口。
4. **一日一买**：每天最多一笔主动买单；成交后当天不再加仓、换股或开新票。
5. **仓位先于观点**：单票初始仓建议不超过10%，同主题风险单元不超过25%；超限时只能减，不能加。
6. **退出机械化**：沿用现有V1基线——初始硬止损最多为买入价下方8%；收盘浮盈达到7%时先兑现一半；首次止盈后，从最高收盘回撤4%继续降风险。若结构失效位更近，以更近者为准。
7. **当天亏损闸门**：账户单日回撤达到1%后停止新增交易；一周达到2.5%后，下一交易日禁止开新仓；当月达到5%后进入5个交易日观察期。
8. **情绪红灯**：出现“必须赚回来”“今天一定要买”“不买就踏空”“再补一点就能回本”中的任何一句，当天自动取消买入资格。

以上仓位和回撤阈值是当前恢复期的保守约束。连续四周执行合格后，再根据实际波动和账户规模调整，不能在盘中临时放宽。

## 五、每天必须完成的三张清单

### 盘前清单

- [ ] 睡眠不少于7小时；若不足，今天不得开新仓。
- [ ] 情绪评分为3/5以上，且没有“急于回本”的念头。
- [ ] 当前持仓、成本、股数、仓位占比和同主题总暴露已确认。
- [ ] 已检查持仓股公告、财报和业绩预告窗口。
- [ ] 每只持仓都写明上涨、震荡、下跌三种情景的动作。
- [ ] 今天允许买入的候选不超过2只，且都在昨天完成研究。

### 下单前清单

- [ ] 当前时间在14:30—14:55。
- [ ] 不是因为突然拉升、讨论热度或急于回本而买。
- [ ] 事件有可靠来源，产业传导和公司映射分别成立。
- [ ] 入场价、失效位、最大亏损和仓位已经写下。
- [ ] 若为加仓，当前持仓已有至少5%收盘浮盈。
- [ ] 交易后单票及同主题仓位不会超限。
- [ ] 今天尚未执行过其他主动买单。

任何一项为“否”，取消订单，不允许解释或通融。

### 盘后清单

- [ ] 所有成交已经录入 `交易记录.xlsx`。
- [ ] 今天发生了什么事实？
- [ ] 我如何解释这些事实？
- [ ] 明天用什么价格、量能或相对强弱验证？
- [ ] 是否出现早盘买入、浮亏加仓、计划外订单或延迟止损？
- [ ] 15:40后关闭行情，不再为当天盈亏寻找新叙事。

## 六、每周安排

### 周五收盘后

只评价执行，不先看盈亏：

- 14:30以前主动买入：每次扣20分。
- 浮亏加仓：每次扣30分。
- 无计划订单：每次扣20分。
- 触发止损后拖延：每次扣30分。
- 漏写盘前或盘后清单：每天扣5分。
- 健身、吃饭或睡前持续盯盘：每天扣5分。

每周从100分开始：

- 80—100分：下一周维持恢复期仓位。
- 60—79分：下一周所有新仓减半。
- 低于60分：下一周禁止开新仓，只做观察和风险处理。

### 周六

- 09:30—11:30：复核一周成交与规则违例，统计而不是自责。
- 14:00—16:00：只做一个行业的深度研究，形成一页结论。
- 其余时间离开市场，正常生活。

### 周日

- 上午完全不看盘。
- 16:00—17:00：建立下周主题优先级，观察池最多5只，计划交易最多2只。
- 17:00以后停止市场信息输入。

## 七、何时才适合真正“以交易为生”

同时满足以下条件前，交易不能承担房租和生活费责任：

- 生活费与交易账户完全隔离，并已有至少12个月基本生活储备。
- 至少连续6个月使用同一套规则实盘，扣除费用后仍稳定为正。
- 连续20个交易日没有早盘追单、浮亏加仓和无计划交易。
- 连续4周执行得分不低于80分。
- 最大回撤始终在事前写定的风险预算内，而不是靠事后扛回来。
- 即使一个月没有盈利，也不会影响房租、吃饭、求职和情绪稳定。

在此之前，交易的定位是“验证系统并积累可重复样本”，求职或其他收入建设必须每天保留至少2小时。
