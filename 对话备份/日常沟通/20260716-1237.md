# 兴业证券交易记录实时接入需求

- 时间：2026-07-16 12:37
- 用户请求：确认兴业证券成交记录如何实时接入个人交易看板。

## 核验结论

- 兴业证券公开公示的零售交易端为优理宝 APP、优理宝智慧版和财富版；官网没有公开面向所有普通个人客户的通用只读 REST API。
- 兴业证券确有官方 SMT-QMT 策略交易终端，但官方将其定位于私募管理人、策略客户、VIP 和高净值客户，账户是否可开通、门槛和接口权限需由客户经理或 95562 确认。
- 迅投官方 XtQuant 文档显示 XtTrader 可查询资产、委托、成交、持仓，并接收这些数据的变动推送。因此若兴业账户获得 SMT-QMT/XtQuant 权限，可作为秒级只读数据源。
- 兴业证券曾公告提醒存在冒充其程序化交易接口的诈骗网站，严禁把账号、交易密码交给第三方网页、抓包脚本或非官方客户端。

## 推荐架构

1. 兴业证券官方 SMT-QMT/MiniQMT 作为券商连接端。
2. 本地只读 Python 适配器订阅成交回报，并定时查询持仓和资产。
3. 原始事件按账户、交易日、委托号、成交号去重后写入本地 SQLite。
4. 看板从 SQLite 读取并通过 WebSocket 更新；交易仍在官方客户端人工完成。
5. 每日收盘后再用交割单或历史成交做一次对账，盘中推送只作为临时状态。
6. 连接中断、数据超时或持仓不一致时，看板亮红灯并禁止生成新买入计划。

## 备选方案

- 若无法申请 QMT：从优理宝导出成交/交割单，由看板监听导入目录，做到手工触发后的分钟级同步；不能称为真正实时。
- 不建议：网页抓包、读取客户端私有数据库、模拟点击登录、保存明文交易密码或使用非官方第三方交易接口。

## 下一步确认清单

联系兴业证券客户经理或 95562，询问账户是否能开通 SMT-QMT/MiniQMT、是否包含 XtQuant Python API、普通股票账户能否查询成交/持仓/资金并接收成交推送、开通门槛与费用、支持的操作系统，以及能否限制为只读使用。

本轮只完成接入方案调研，没有连接券商账户，也没有修改看板代码。
